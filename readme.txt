== Description ==

Import suppliers, products, customers and orders from Reviso to your Woocommerce store. 

**AVAILABLE FEATURES**

* Import clients from Reviso as WordPress users.
* Import suppliers from Reviso as WordPress users.
* Import orders from Reviso.
* Import products from Reviso.

== Installation ==

* Upload the 'wc-importer-for-reviso-premium’ directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
* Activate Reviso for WooCommerce from your Plugins page.
* Once Activated, go to WooComerce/ WC Importer for Reviso menu and set you preferences.
